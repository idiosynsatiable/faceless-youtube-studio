# Shorts algorithm strategy — legitimate signals only

Faceless YouTube Studio recommends only legitimate algorithm-friendly behavior. The platform rewards retention, not tricks. The system never recommends bot engagement, paid pods, or any tactic that violates YouTube terms.

## Levers we recommend

- Retention: average view duration ratio, swiped-away rate, completion rate
- Hooks: pattern interrupt within 0-2 seconds, clear promise within 7 seconds
- Niche consistency: topic clusters that share the same audience promise
- Topic clusters: 4-6 evergreen clusters per niche, rotated every week
- Useful captions: burned-in caption lines per beat, accessibility-compliant contrast
- Pacing: cuts on motion every 1.5-3 seconds, no audio gaps that drift below the narration
- Upload rhythm: same publish windows daily, sustainable cadence over volume
- Shorts-to-long-form connection: pinned comment + description + end-frame CTA
- Audience comments: respond within 48 hours, pin highest-value reply
- Analytics-driven iteration: weekly review of the worst Short to identify why it lost retention
- Strong first frame: single subject, no clickbait overlay
- Fast scene changes when the format calls for them; slow holds when narration calls for them
- Playlist and channel structure: clusters become playlists; trailers reinforce the niche promise

## Levers we never recommend

- Buying subscribers, likes, views, comments
- Bot networks, comment bots, view bots, engagement pods
- Sub-for-sub schemes
- Misleading thumbnails or titles
- Irrelevant hashtags or deceptive tags
- Mass low-effort uploads or duplicate variants
- Stolen or unlicensed content compilations
- Spam links in comments or DMs
- Impersonation of creators, brands, or experts

## Cadence rule

When the user asks for "as many as needed," the cadence engine interprets that as the most aggressive sustainable schedule that maintains quality and respects audience fatigue signals — never a license to spam.

## Audience fatigue handling

The cadence engine flags a topic cluster that appears 5 or more times in a window and recommends rotating angles or pausing for 48 hours. If audience fatigue signal is `high`, the engine reduces per-day cadence by one and suggests a different cluster.
